# Mini Gestor de Tareas (Kotlin) - Taller

## Descripción
Aplicación de un **mini gestor de tareas**. Funciones:
- Agregar, listar, marcar/desmarcar, eliminar y editar tareas.
- Búsqueda por texto (case-insensitive).
- Prevención de títulos duplicados.
- Persistencia simple en `tasks.txt` (archivo de texto con formato `id;title;done`).
- Exportar a CSV (`tasks_export.csv`).
- Estadísticas rápidas.
